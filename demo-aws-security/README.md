🔐 AWS Security Demo – Terraform
This repository contains a simple demo to showcase core AWS security practices using Terraform.

The goal is to demonstrate how to protect, monitor, and detect threats in a cloud environment using native AWS services.

## 🚀 What this demo covers

This demo includes:

### Secure S3 Bucket
- Versioning enabled
- Server-side encryption
- Block public access

### CloudTrail
Multi-region logging
API activity tracking
GuardDuty
Threat detection (suspicious behavior, compromised credentials)
Security Hub
Centralized security findings
AWS best practices checks
Security Groups
Example of insecure configuration (0.0.0.0/0)
Example of secure configuration (restricted access)

## 📦 Requirements
- Terraform >= 1.0
- AWS CLI configured
- AWS account with permissions to create resources

## ⚠️ Disclaimer
This is a demo project for educational purposes only.
Do not use in production without proper customization and security review.
